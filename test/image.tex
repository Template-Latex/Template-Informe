\section{insertimage + variantes}

Este archivo prueba las diferentes funciones para insertar figuras.

\insertimage{\img}{width=\w}{insertimage normal}

\insertimageboxed{\img}{width=\w}{5}{insertimage normal con caja}

Sin números:


\insertimage{\img}{width=\w}{}

\insertimageboxed{\img}{width=\w}{5}{}

Ahora con links:

\insertimagelink{\img}{width=\w}{\link}{insertimage normal}

\insertimagelinkboxed{\img}{width=\w}{5}{\link}{insertimage normal con caja}

Con links, sin números:

\insertimagelink{\img}{width=\w}{\link}{}

\insertimagelinkboxed{\img}{width=\w}{5}{\link}{}


\section{Objeto images}

\begin{images}{Ejemplo collage}

	\addimageanumboxed{\img}{width=\wc}{5}
	
	\imagesnewline
	
	% Con links
	\addimagelink{\img}{width=\wc}{\link}{Example}
	\addimageanumlink{\img}{width=\wc}{\link}
	\addimagelinkboxed{\img}{width=\wc}{5}{\link}{Example}
	\addimageanumlinkboxed{\img}{width=\wc}{5}{\link}
\end{images}
